# /youtube-script — 유튜브 영상 대본 완전 작성

주제와 타겟 시청자를 입력받아 유튜브 알고리즘 최적화와 시청자 리텐션을 극대화하는 완전한 영상 대본을 작성한다. 제목, 설명, 태그, 챕터 타임코드, 자막 파일까지 포함한다.

## 실행 단계

1. **영상 기획**
   - `$ARGUMENTS`에서 주제, 영상 길이, 타겟 시청자 파싱
   - `C:/oomni-data/research/` 최신 리서치에서 관련 인사이트 수집
   - 경쟁 영상 분석 (같은 주제 상위 5개 영상 제목/구조 파악)
   - 영상 포맷 결정: 튜토리얼/스토리텔링/리뷰/인터뷰/다큐

2. **제목 후보 생성**
   알고리즘 최적화 제목 10개:
   - CTR 높은 패턴: "X가지", "하는 법", "비밀", "진실", "실패 이유"
   - 감정 자극: 호기심, 두려움 해소, 성취욕
   - 길이: 40-60자 (썸네일 텍스트와 분리)

   썸네일 텍스트 3가지도 함께 생성 (7단어 이내)

3. **훅 섹션 작성 (0-30초)**
   시청자 이탈 방지를 위한 강력한 오프닝:
   - Pattern Interrupt (예상과 다른 시작)
   - 결과 먼저 보여주기
   - 충격적인 통계나 사실
   - 공감 유발 질문

4. **대본 구조 설계**
   영상 길이별 구조:
   - **10분 영상**: 훅(30s) → 소개(60s) → 본론 3섹션(각 2-3분) → 요약(60s) → CTA(30s)
   - **20분 영상**: 훅(45s) → 소개(90s) → 본론 5섹션 → 요약(90s) → CTA(60s)

5. **대본 작성**
   각 섹션별:
   - 실제 말할 내용 그대로 (구어체)
   - [화면 설명] 브라켓으로 B-roll 지시
   - [강조] 중요 단어 표시
   - (편집 노트: ...) 편집 지시사항
   - 자연스러운 전환 문구

6. **자막 파일 생성**
   - SRT 포맷 (YouTube 자막 업로드용)
   - VTT 포맷 (웹 플레이어용)
   - 한국어 자막 기본, 영어 자막 선택

7. **YouTube 최적화 메타데이터 생성**
   - 최종 제목 1개
   - 설명란 (처음 150자 중요 + 챕터 타임코드 + 링크 + 해시태그)
   - 태그 500자 이내
   - 챕터 타임코드 (00:00 형식)
   - 끝 화면 CTA 텍스트

8. **커뮤니티 포스트 생성**
   영상 업로드 후 활용할 유튜브 커뮤니티 게시글

## 출력 형식

### 대본 파일 (`youtube-script_YYYY-MM-DD.md`)

## 추가 인자

`$ARGUMENTS` — 다음 형식으로 입력:
- `[주제] [영상 타입] [목표 길이(분)]`
- `--format tutorial|story|review|documentary` : 영상 형식
- `--length 5|10|20` : 목표 영상 길이 (분)
- `--bilingual` : 한/영 자막 동시 생성
- `--no-broll` : B-roll 지시사항 제외
- `--tone casual|educational|documentary` : 톤
- 예시: `/youtube-script "AI 자동화 30일 실험" story 12 --tone casual --bilingual`
