핵심 기능

AI 회사를 경영하다

실제 팀원처럼 일하는 AI 봇, 당신은 방향만 잡으세요.

봇-to-봇 체인 자동화

Research Bot 완료 → Content Bot 자동 시작. 봇이 서로 결과를 넘기며 파이프라인을 만들어냅니다.

승인 인박스

봇이 결정 못하는 것만 모아서 보여줍니다. 클릭 하나로 처리. 중요한 결정은 항상 당신이.

비용 추적 & 예산

봇별 토큰 사용량과 비용을 실시간 추적. 예산 한도 초과 시 자동 일시정지.

PIN 잠금 · 로컬 보안

앱 PIN 잠금, AES-256 자격증명 암호화, 데이터는 내 PC에만. 클라우드 없음.

외부 서비스 연동

Slack, Notion, Gmail, Stripe, GitHub, Linear 등 10종 연동. 웹훅으로 외부 트리거도 가능.

자동 리포트

일간/주간/월간 운영 리포트 자동 생성. CEO Bot이 전체 봇 활동을 종합해 보고합니다.

사용법

5분이면 시작됩니다

설치부터 첫 봇 실행까지.

1

OOMNI 설치

아래 다운로드 버튼을 눌러 OOMNI Setup.exe를 받고 더블클릭으로 설치. macOS는 DMG 파일을 Applications로 드래그.

2

미션 입력 + API 키 등록

지금 만드는 서비스를 한 줄로 입력하고 Anthropic API 키를 붙여넣으세요. console.anthropic.com에서 무료 발급 가능.

3

봇 추가 후 실행

원하는 봇을 추가하고 실행 버튼을 누르세요. 봇이 알아서 일하고 실시간 피드에 결과를 보고합니다.

승인만 하면 끝

봇이 확인을 요청할 때만 알림. 승인/거절 클릭 하나로 처리. 나머지는 AI가 이어서 자동 진행.
