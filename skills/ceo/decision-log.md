# /decision-log — 중요 비즈니스 결정 기록 및 아카이빙

가격 변경, 기능 추가/제거, 파트너십, 채용, 피벗 등 중요한 비즈니스 결정을 구조화된 형식으로 기록한다. 결정의 배경, 대안, 예상 결과, 실제 결과를 추적하여 미래 의사결정을 개선하는 학습 데이터베이스를 구축한다.

## 실행 단계

1. **결정 유형 분류**
   `$ARGUMENTS`에서 결정 내용 파싱, 유형 자동 분류:
   - `pricing` — 가격 정책 변경
   - `feature` — 기능 추가/제거/변경
   - `strategy` — 전략적 방향 전환
   - `ops` — 운영 프로세스 변경
   - `hr` — 채용/외주/팀 구성
   - `partnership` — 파트너십/통합
   - `marketing` — 마케팅 채널/메시지 변경
   - `tech` — 기술 스택/아키텍처 결정

2. **결정 배경 기록**
   - 어떤 문제/기회를 해결하려는가
   - 현재 상황 (데이터로 뒷받침)
   - 결정을 강제한 외부 요인 (마감, 경쟁사, 사용자 피드백)

3. **고려한 대안 목록**
   각 대안에 대해:
   - 장점
   - 단점
   - 왜 선택하지 않았는가

4. **최종 결정 상세**
   - 결정한 내용 (명확하게)
   - 결정 근거 (1-3가지)
   - 예상되는 결과 (수치 목표 포함)
   - 성공 측정 기준 (30일, 90일)

5. **리스크 및 대응 계획**
   - 결정이 틀렸을 때의 신호
   - 롤백 또는 피벗 조건

6. **결정 로그 파일에 추가**
   기존 결정 로그 파일에 새 항목 추가:
   `C:/oomni-data/ceo/decision-log.md`

7. **30일/90일 리뷰 알림 설정**
   n8n을 통해 Telegram 리마인더:
   - D+30: "결정 #DEC-XXX 30일 리뷰 시점"
   - D+90: "결정 #DEC-XXX 90일 결과 기록 필요"

8. **과거 결정 리뷰** (`--review` 옵션)
   30일/90일이 지난 결정에 대해:
   - 예상 vs 실제 결과 비교
   - 결정 품질 평가 (1-5점)
   - 다음에 다르게 할 것

## 출력 형식

### 의사결정 로그 (누적 파일 `decision-log.md`)

```markdown
# OOMNI 의사결정 로그

> 마지막 업데이트: YYYY-MM-DD

---

## DEC-2026-018 — 프리미엄 플랜 가격 ₩49,000 → ₩59,000 인상

**날짜**: 2026-04-05
**유형**: pricing
**결정자**: 장우경
**상태**: 실행됨 ✅
**리뷰 예정**: 2026-05-05 (30일), 2026-07-05 (90일)

### 배경

현재 Pro 플랜 ₩49,000은 경쟁사 평균(₩65,000) 대비 24% 낮음.
ARPU ₩28,736 → ₩35,000 달성을 위해 가격 조정 필요.
지난 30일 이탈 분석 결과 가격 언급 1건 (전체 이탈 사유의 3%).

**근거 데이터**:
- LTV:CAC = 19.2x (건강하므로 인상 여지 있음)
- 무료 → 유료 전환 시 가격 이탈: 3% (낮음)
- 경쟁사 평균: ₩65,000

### 고려한 대안

| 대안 | 장점 | 단점 | 채택 여부 |
|------|------|------|---------|
| 현행 유지 (₩49,000) | 이탈 위험 없음 | ARPU 목표 달성 어려움 | ❌ |
| ₩55,000 인상 | 점진적 조정 | 임팩트 작음 | ❌ |
| ₩59,000 인상 | 경쟁사와 합리적 격차 | 단기 이탈 5-10% 가능 | ✅ 채택 |
| ₩69,000 인상 | ARPU 극대화 | 이탈 위험 높음 | ❌ |

### 최종 결정

Pro 플랜을 ₩49,000 → ₩59,000으로 인상.
기존 고객은 3개월 유예 (2026-07-05까지 기존 가격 유지).
신규 가입자는 즉시 적용.

**근거**:
1. 경쟁사 대비 여전히 9% 저렴 (정당성 확보)
2. 기존 고객 유예로 단기 이탈 최소화
3. ARPU ₩8,000 상승 예상 → MRR +₩688,000 (86명 기준)

### 예상 결과

| 지표 | 현재 | 30일 후 예상 | 90일 후 예상 |
|------|------|------------|------------|
| Pro 구독자 | 32명 | 29명 (-9%) | 33명 |
| ARPU | ₩28,736 | ₩31,000 | ₩33,500 |
| MRR | ₩2,480,000 | ₩2,590,000 | ₩2,750,000 |

### 리스크 및 대응

- **위험 신호**: 30일 내 Pro 이탈 15% 이상
- **대응**: 가격 롤백 + 기존 가격 6개월 연장 제공

---

## DEC-2026-017 — 30일 리뷰 결과

**원래 결정**: [DEC-2026-015] 트위터를 주 마케팅 채널로 집중

| 예상 | 실제 | 평가 |
|------|------|------|
| 팔로워 +200명/월 | +340명/월 | ✅ 초과 달성 |
| 유입 트래픽 +30% | +47% | ✅ 초과 달성 |

**결정 품질 점수**: 5/5
**배운 점**: 스레드 형식이 단일 트윗보다 3배 높은 팔로워 증가. 다음에도 스레드 우선.

---
```

## 추가 인자

`$ARGUMENTS` — 다음 형식으로 입력:
- `[결정 내용 설명]`
- `--type pricing|feature|strategy|ops|hr|partnership|marketing|tech` : 결정 유형
- `--review DEC-2026-015` : 기존 결정 리뷰 모드 (실제 결과 기록)
- `--list` : 리뷰 기한이 지난 결정 목록 보기
- `--remind` : 리뷰 예정 결정 Telegram 발송
- 예시: `/decision-log "Pro 플랜 가격 ₩49,000 → ₩59,000 인상" --type pricing`
