왜 OOMNI인가

n8n, Zapier 세팅에 지쳐본 적 있나요? AI가 알아서 다 합니다.

⚡

API 키 1개로 끝

Anthropic API 키 하나만 있으면 시작. Firebase, PostgreSQL 설치 불필요. 왕초보도 5분 내 첫 봇 실행.

🤖

봇이 알아서 협업

Research Bot이 끝나면 Content Bot이 자동으로 깨어납니다. 승인만 누르면 Growth Bot이 배포까지 처리.

🔒

100% 로컬 실행

데이터가 클라우드에 올라가지 않습니다. SQLite로 내 PC에만 저장. 월정액 없음, 서버비 없음.

✅

승인 인박스

봇이 혼자 결정 못하는 것만 모아서 보여줍니다. 클릭 하나로 승인/거절. 중요한 결정은 항상 사람이.

💰

비용 추적

봇별 AI 토큰 사용량과 비용을 실시간 추적. 예산 한도 설정으로 과금 걱정 없음.

📊

자동 리포트

일간/주간/월간 운영 리포트 자동 생성. CEO Bot이 전체 봇 활동을 종합해서 보고합니다.
