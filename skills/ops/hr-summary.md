# /hr-summary — 외주/프리랜서 인력 관리 및 정산 요약

외주 개발자, 프리랜서 디자이너, 콘텐츠 작성자 등 외부 인력의 작업 내역, 정산 현황, 계약 상태를 정리하고 다음 달 인력 계획을 수립한다. 원천세 신고 준비도 포함한다.

## 실행 단계

1. **인력 데이터 로드**
   `C:/oomni-data/ops/hr/contractors.json` 읽기:
   ```json
   {
     "contractors": [
       {
         "id": "c001",
         "name": "홍길동",
         "role": "프론트엔드 개발",
         "rate": 50000,
         "rate_unit": "hourly",
         "contract_end": "2026-06-30",
         "payment_method": "계좌이체",
         "withholding_tax": true
       }
     ]
   }
   ```
   없으면 `$ARGUMENTS`에서 인력 정보 직접 입력.

2. **이번 달 작업 내역 집계**
   각 외주 인력에 대해:
   - GitHub 커밋/PR 수 (개발자)
   - Figma 파일 업데이트 (디자이너)
   - 작업물 파일 수 (콘텐츠)
   - 청구된 시간/건수
   - 제출된 인보이스 확인

3. **정산 계산**
   각 인력별:
   - 청구 금액 (세전)
   - 원천세 3.3% (프리랜서/사업소득 공제)
   - 실 지급 금액 (세후)
   - 지급 예정일

4. **계약 현황 점검**
   - 계약 만료 임박 (30일 이내): 갱신 여부 결정 필요
   - 계약 미체결 상태로 작업 중인 인력 경고
   - NDA 체결 여부 확인

5. **원천세 납부 준비**
   매월 10일까지 납부해야 하는 원천세:
   - 이번 달 지급액 합계
   - 원천세 합계 (3.3%)
   - 납부 기한 알림

6. **인력 성과 평가**
   작업 품질 기록 (주관적 평가 + 객관적 지표):
   - 마감 준수율
   - 수정 요청 횟수
   - 커뮤니케이션 평점

7. **다음 달 인력 계획**
   - 예정된 개발 작업 → 필요 인력 계산
   - 예산 한도 내 가능한 외주 시간 계산
   - 새 인력 채용 필요 여부

## 출력 형식

### HR 요약 보고서 (`hr-summary_YYYY-MM.md`)

```markdown

**원천세 납부 예정**: ₩34,650 (납부 기한: YYYY-MM-10)

---

| 홍길동 | 프론트개발 | 20시간 | ₩600,000 | ₩19,800 | ₩580,200 | ✅ 정산완료 |
| 김민지 | UI 디자인 | 건당 | ₩300,000 | ₩9,900 | ₩290,100 | ⏳ 인보이스 검토 중 |

**이번 달 원천세 납부액**: ₩34,650
**납부 기한**: YYYY-MM-10
**납부 방법**: 홈택스 → 원천세 → 반기별 납부 (또는 매월)

---

## 계약 현황

| 이름 | 계약 종료일 | D-day | 갱신 여부 |

## 다음 달 계획

**예산**: ₩1,200,000

- 결제 페이지 UI 개선 (홍길동, 예상 25시간 = ₩1,250,000) ← ⚠️ 예산 초과

**예산 조정 필요**: 홍길동 작업 범위 축소 또는 다음 달로 이월
```

## 저장 위치

- `C:/oomni-data/ops/hr/hr-summary_YYYY-MM.md`
- `C:/oomni-data/ops/hr/hr-payroll_YYYY-MM.json`
- `C:/oomni-data/ops/hr/contractors.json` (인력 마스터 데이터, 지속 업데이트)

## 추가 인자

`$ARGUMENTS` — 선택적으로 다음을 지정할 수 있다:
- `--month YYYY-MM` : 특정 월 (기본값: 이번 달)
- `--add-contractor` : 새 외주 인력 추가 (인터랙티브)
- `--export excel` : Excel 호환 CSV로 내보내기
- `--withholding-tax` : 원천세 납부 신고서 양식 생성
- 예시: `/hr-summary --month 2026-04 --withholding-tax`
