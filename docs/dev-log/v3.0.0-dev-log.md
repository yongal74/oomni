## 수정 이력

### BotRunModal 테스트 수정

**문제**: `useNavigate()` requires Router context
```tsx
// 수정 전
render(<BotRunModal agent={mockAgent} onClose={onClose} />)
// → Error: useNavigate() may be used only in the context of a <Router>

// 수정 후
function renderWithRouter(ui) {
  return render(<MemoryRouter>{ui}</MemoryRouter>)
}
```

**추가 수정**: BotRunModal 동작이 `agentsApi.trigger` 호출 방식에서
`navigate('/dashboard/bots/:id?autorun=...')` 방식으로 변경됨 → 테스트 로직 재작성

### PipelineBar 테스트 수정

**문제**: PipelineBar props가 `role: string`이 아닌 `stages: Stage[]` 형식
```tsx
// 수정 전 (잘못된 타입)
<PipelineBar role="research" currentStage={null} />

// 수정 후 (올바른 props)
const stages = ROLE_STAGES['research']!
<PipelineBar stages={stages} currentStage={null} />
```

### claudeCodeService 테스트 수정

**문제**: spawn args에서 task 위치 오판
```typescript
// 수정 전
const taskArg = capturedArgs[capturedArgs.length - 1]

// 수정 후: --print 플래그 다음 위치
const printIdx = capturedArgs.indexOf('--print')
expect(printIdx).toBeGreaterThan(-1)
const taskArg = capturedArgs[printIdx + 1]
```
