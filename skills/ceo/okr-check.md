# /okr-check — 분기 OKR 달성률 체크 및 전략 조정

현재 분기의 OKR 진행 상황을 모든 지표 데이터와 비교하여 달성률을 계산하고, 목표 대비 Gap 분석과 잔여 기간 내 달성 전략을 수립한다.

## 실행 단계

1. **현재 OKR 로드**
   `C:/oomni-data/ceo/okr_YYYY-QN.json` 읽기:
   ```json
   {
     "quarter": "2026-Q2",
     "objectives": [
       {
         "id": "O1",
         "title": "프로덕트-마켓 핏 강화",
         "key_results": [
           { "id": "KR1.1", "title": "D7 리텐션 50% 달성", "target": 50, "unit": "%", "current": 41 },
           { "id": "KR1.2", "title": "NPS 40 달성", "target": 40, "unit": "점", "current": 32 }
         ]
       }
     ]
   }
   ```
   없으면 `$ARGUMENTS`에서 OKR 직접 입력.

2. **현재 지표 수집**
   각 KR의 현재값을 다음 소스에서 자동 조회:
   - MRR, DAU, 리텐션: `growth/weekly-report_YYYY-WW.json` (최신)
   - 콘텐츠 KPI: `content/` 디렉터리 최신 파일
   - 코드 KPI: git 통계
   - 비용 KPI: `ops/costs/costs_YYYY-MM.json` (최신)

3. **달성률 계산**
   각 KR에 대해:
   - 달성률 (%) = (현재값 - 시작값) / (목표값 - 시작값) × 100
   - 목표 달성 여부 예측 (현재 진행 속도 기준)
   - 경과 기간 대비 진행률 (시간 기반 기대치)

4. **신호등 상태 분류**
   각 KR에 신호등 표시:
   - 🟢 Green: 목표의 70% 이상 진행 (또는 기대치 상회)
   - 🟡 Yellow: 목표의 50-70% 진행
   - 🔴 Red: 목표의 50% 미만 진행 (위험)

5. **Objective별 건강도 계산**
   소속 KR들의 평균 달성률로 Objective 건강도 산출

6. **Gap 분석**
   목표 미달 KR에 대해:
   - 현재 Gap (수치)
   - 잔여 기간 내 필요한 주간 성장률
   - 현실적 달성 가능성 판단

7. **전략 조정 제안**
   각 Red/Yellow KR에 대해:
   - 즉시 실행 가능한 액션 3가지
   - 목표값 하향 조정 필요 여부 판단
   - 다른 KR과의 상충 관계 분석

8. **OKR 업데이트 파일 저장**
   현재 달성률로 JSON 업데이트

9. **분기 마감 예측**
   현재 추세 유지 시 분기 말 예상값:
   - 낙관적/현실적/보수적 3가지 시나리오

## 출력 형식

## Gap 분석 및 액션 플랜

### 🔴 KR2.3 이탈률 3% 이하 (현재: 4.6%, Gap: 1.6%p)
**잔여 기간**: 8주
**필요 주간 개선**: 0.2%p/주

**즉시 실행 액션**:
1. At-Risk 세그먼트 재활성화 이메일 이번 주 발송
2. 이탈 고객 5명 인터뷰 예약 (가장 빠른 인사이트 획득)
3. 이탈 원인 Top 3 파악 후 온보딩 수정

**현실적 달성 가능성**: 4.0% (목표 3% 는 현 추세에서 어려움)
**목표 재조정 제안**: 3% → 3.5% (보수적 유지 시)

---

## 추가 인자

`$ARGUMENTS` — 선택적으로 다음을 지정할 수 있다:
- `--quarter YYYY-QN` : 특정 분기 (기본값: 현재 분기)
- `--update KR1.1:45` : 특정 KR 수동 업데이트
- `--scenario` : 분기 말 시나리오 분석 포함
- `--adjust` : 달성 어려운 KR 목표 재조정 제안
- `--send` : Telegram으로 요약 발송
- 예시: `/okr-check --quarter 2026-Q2 --scenario --send`
