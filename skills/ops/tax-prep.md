# /tax-prep — 종합소득세 / 부가세 신고 준비 자동화

월간/분기 재무 데이터를 기반으로 세금 신고에 필요한 서류를 자동으로 정리한다. 부가세 (분기별), 종합소득세 (연간), 전자세금계산서 발행 내역을 집계하여 세무사 전달용 패키지를 생성한다.

## 실행 단계

1. **신고 유형 및 기간 결정**
   - `$ARGUMENTS`에서 세금 유형(vat/income) 및 기간 파싱
   - 부가세: 1월 신고 (7-12월분), 7월 신고 (1-6월분)
   - 종합소득세: 5월 신고 (전년도)
   - 법인세: 3월 신고 (전년도)

2. **수익 데이터 집계**
   해당 기간의 `monthly-finance_YYYY-MM.json` 파일 전체 읽기:
   - 총 수익 (국내/해외 분리)
   - 환불/취소 금액
   - Stripe USD 수익 → KRW 환산 (해당 월 기준 환율)
   - 전자세금계산서 발행 내역 (기업 고객)

3. **비용 데이터 집계**
   - 적격 증빙 있는 비용 목록 (세금계산서, 현금영수증, 카드 영수증)
   - 비용 종류별 분류: 소모품비, 통신비, 지급수수료, 광고비
   - 해외 결제 비용 (USD) → KRW 환산

4. **부가세 계산**
   ```
   매출세액 = 국내 과세 매출 × 10%
   - 매입세액 = 국내 구입 비용 × 10% (영수증 있는 것만)
   = 납부할 부가세
   ```
   - 해외 결제 서비스 수익 = 영세율 (0%)
   - 수출 영세율 적용 여부 확인

5. **종합소득세 계산** (연간 신고)
   ```
   총 수입금액
   - 필요경비
   = 사업소득금액
   - 소득공제
   = 과세표준
   × 세율 (6~45% 누진세율)
   = 산출세액
   - 세액공제
   = 납부할 세액
   ```
   - 국민연금, 건강보험료 공제
   - 청년 창업 감면 여부 확인
   - 예납세액 차감

6. **전자세금계산서 발행 내역 정리**
   - 기업 고객에게 발행한 세금계산서 목록
   - 미발행 건 감지 (기업 고객 중 세금계산서 미발행)
   - 수정세금계산서 발행 필요 건

7. **세무사 전달 패키지 생성**
   - 수익/지출 요약 Excel 호환 CSV
   - 카드 사용 내역 CSV
   - 세금계산서 목록 CSV
   - 체크리스트 (세무사에게 추가 확인 요청사항)

8. **세무 캘린더 알림 설정**
   주요 신고 기한 목록과 D-14, D-7, D-1 Telegram 알림 설정

## 출력 형식

### 세금 신고 준비 보고서 (`tax-prep_[타입]_YYYY-[기간].md`)

```markdown

**신고 기한**: 2026년 7월 25일 (D-20)
**추정 납부세액**: ₩238,200

---

## 수익 집계

| 월 | 국내 과세 매출 | 해외 영세율 | 합계 |
|----|-------------|-----------|------|

| 2월 | ₩2,100,000 | $220 (₩297,000) | ₩2,397,000 |

## 부가세 계산

```

= 납부 예상 부가세:                  ₩1,286,000

## 추가 인자

`$ARGUMENTS` — 선택적으로 다음을 지정할 수 있다:
- `--type vat|income|corporate` : 세금 종류 (기본값: vat)
- `--period 1h|2h|annual` : 신고 기간 (1기/2기/연간)
- `--year 2026` : 대상 연도
- `--business sole|corp` : 사업 형태 (개인사업자/법인)
- `--export csv` : 세무사 전달용 CSV 패키지 생성
- 예시: `/tax-prep --type vat --period 1h --year 2026 --export csv`
