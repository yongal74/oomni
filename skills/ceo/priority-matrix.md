# /priority-matrix — 임팩트/긴급도 우선순위 결정 매트릭스

현재 적체된 태스크, 아이디어, 프로젝트를 입력받아 2×2 매트릭스(아이젠하워 매트릭스 + ICE 스코어)로 분류하고, 이번 주 집중해야 할 TOP 5 태스크를 확정한다.

## 실행 단계

1. **현재 태스크 목록 수집**
   다음 소스에서 미완료 태스크 수집:
   - `C:/oomni-data/ceo/task-backlog.md` 읽기 (없으면 생성)
   - `$ARGUMENTS`에서 새 태스크 추가
   - `C:/oomni-data/ceo/weekly-brief_YYYY-WW.json`의 `next_week_priorities` 읽기
   - git issues 또는 TODO 주석 스캔

2. **각 태스크 평가**
   5개 차원에서 1-10점 스코어링:
   - **Impact (영향도)**: 비즈니스 목표(MRR, 성장)에 미치는 영향
   - **Urgency (긴급도)**: 지금 안 하면 발생하는 페인
   - **Confidence (확신도)**: 이 태스크가 효과 있을 확신
   - **Ease (용이도)**: 구현 난이도의 역수 (쉬울수록 높은 점수)
   - **Dependency (의존성)**: 다른 태스크가 이것에 의존하는 정도

3. **2×2 매트릭스 분류**
   아이젠하워 기준:
   - **Q1 (긴급 + 중요)**: 즉시 실행 — DO NOW
   - **Q2 (긴급하지 않음 + 중요)**: 일정 잡기 — SCHEDULE
   - **Q3 (긴급 + 중요하지 않음)**: 위임/자동화 — DELEGATE
   - **Q4 (긴급하지 않음 + 중요하지 않음)**: 제거 — ELIMINATE

4. **ICE 스코어 계산**
   `ICE = Impact × Confidence × Ease`
   ICE 점수 내림차순으로 정렬

5. **이번 주 집중 TOP 5 선정**
   Q1 항목 우선, Q2 중 ICE 높은 것 추가
   각 태스크에 대해:
   - 예상 소요 시간
   - 완료 기준 (Definition of Done)
   - 다음 단계 (첫 번째 액션)

6. **위임/자동화 대상 목록**
   Q3 항목 중 n8n으로 자동화 가능한 것 표시

7. **제거 대상 목록**
   Q4 항목을 공식적으로 "안 하기로 결정" 처리

8. **태스크 백로그 업데이트**
   `C:/oomni-data/ceo/task-backlog.md` 업데이트:
   - 완료된 태스크 체크
   - 새 태스크 추가
   - 우선순위 업데이트

## 2×2 매트릭스

```
                    [긴급]
                      │
                      │
  Q1: 즉시 실행       │   Q3: 위임/자동화
  ─────────────────   │   ────────────────
  [중요] ● 온보딩 개선 │   ● 소셜 미디어 예약
         ● 비용 최적화 │   ● 리서치 보고서 발송
         ● PH 준비    │
  ─────────────────   │   ────────────────
  Q2: 일정 잡기       │   Q4: 제거
  ─────────────────   │   ────────────────
  ● 아키텍처 문서화   │   ● 새 디자인 시스템 (너무 이름)
  ● 멀티테넌시 설계   │   ● 팟캐스트 채널 개설
  [중요하지 않음]     │
                      │
                [긴급하지 않음]
```

## 추가 인자

`$ARGUMENTS` — 다음 형식으로 입력:
- 새 태스크 목록 (줄바꿈 또는 쉼표 구분)
- `--add "태스크 이름"` : 새 태스크 추가
- `--complete "태스크 ID"` : 완료 처리
- `--focus mrr|growth|debt` : 집중 영역 (MRR 향상/성장/기술부채)
- `--hours-budget 40` : 이번 주 가용 시간 (기본값: 40)
- 예시: `/priority-matrix --add "Product Hunt 런치 준비" --focus mrr --hours-budget 35`
